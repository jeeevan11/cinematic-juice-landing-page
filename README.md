# Cinematic Juice Landing Page

An ultra-premium, scroll-driven cinematic landing page for a cold-pressed juice brand. This project features stunning 3D elements, smooth scrolling, and advanced animations to deliver an award-winning user experience.

## ✨ Features

- **Smooth Scrolling** powered by Lenis
- **Advanced Animations** timeline and scroll-driven physics using GSAP
- **3D Renderings** using Three.js / React Three Fiber
- **Modern Styling** with Tailwind CSS v4
- **Optimized Fast Builds** using Vite

## 🚀 Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed (version 18+ recommended).

### Installation

1. Clone this repository (if you haven't already):
   ```bash
   git clone https://github.com/jeeevan11/cinematic-juice-landing-page.git
   cd cinematic-juice-landing-page
   ```

2. Install the project dependencies:
   ```bash
   npm install
   ```

### Running the Development Server

To start the local development server:

```bash
npm run dev
```

Then, open your browser and navigate to `http://localhost:5173`. 

## 🛠️ Tech Stack

- **Framework:** React 19
- **Build Tool:** Vite 8
- **Styling:** Tailwind CSS 4
- **Animations:** GSAP & Lenis Setup
- **3D Graphics:** Three.js

## 📦 Build for Production

To create an optimized production build, run:
```bash
npm run build
```

To preview the production build locally:
```bash
npm run preview
```
